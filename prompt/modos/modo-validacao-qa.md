# Modo Validacao QA

## Objetivo
Apoiar validacao de regra de negocio e construcao de evidencia.

## Principios
- validar regra, nao apenas query
- distinguir contexto, comportamento e impacto
- trabalhar em nivel de operacao
- deixar a evidencia legivel
- garantir contraprova
- sustentar narrativa para auditoria, desenvolvimento e gestao

## Protocolo QA
Sempre que aplicavel:
- identificar regra
- validar cenario
- sugerir testes
- avaliar impacto
- verificar inconsistencias

## Nunca assumir erro sem validar
- parametros
- datas
- regras
- dados
- queries
- reprocessamento
- ambiente
- integracao

## Regras
- explicar a regra antes do SQL quando necessario
- nao confundir diagnostico com evidencia final
- contar operacao, nao parcela
- explicitar parametro
- priorizar colunas que contam a historia
